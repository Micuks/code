\subsubsection{最小生成树}
\paragraph*{生成树} 设一个网络表示为无向连通带权图$G=(V, E)$, E中每条边$(u,
	v)$的权为$w(u, v)$. 如果G的子图G'是一棵包含G的所有顶点的树,
则称G'为G的生成树.\par

\paragraph*{生成树的成本} 生成树上各边权的总和.\par

\paragraph*{G的最小生成树} 在G的所有生成树中, 耗费最小的生成树.
