本节介绍对cbwd中值为"cv"的单元格使用后项数据填充.

\subsubsection{实现介绍}
关键代码段如下:
\begin{python}
    for i in range(len(cbwd.index), 1, -1):
        if cbwd[i-1] == 'cv':
            cnt += 1
        cbwd[i-1] = cbwd[i] if cbwd[i-1] == "cv" else cbwd[i-1]
\end{python}
由于是从后往前替换, 所以即便是多个相连的cv, 也能完成替换.
